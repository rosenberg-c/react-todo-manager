# Go + HTMX Todo Manager (POC)

A todo manager built with Go, HTMX, and templ.

## Prerequisites

- Go 1.21+
- Node.js 18+

## Quick Start

```bash
# Install tools and dependencies
make tools
make deps

# Run development server (hot reload)
make dev
```

Open http://localhost:8080

## Commands

| Command | Description |
|---------|-------------|
| `make dev` | Run with hot reload |
| `make test` | Run all tests (Go + e2e) |
| `make fmt` | Format code |
| `make css-watch` | Watch CSS for changes |
